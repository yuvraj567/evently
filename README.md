# 🎉 Full Stack Event Platform with Next.js 14

A modern, full-stack event management web app built with **Next.js 14**, **TypeScript**, and **Stripe**, inspired by platforms like **Eventbrite** and **Meetup**.

## 📦 Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS + Shadcn UI
- MongoDB + Mongoose
- Clerk (Authentication)
- Stripe (Payments)
- Zod (Validation)
- React Hook Form
- Uploadthing (File Uploads)

## 🚀 Features

- 🔐 User Auth (Login, Register, Protected Routes)
- 📅 Create, Read, Update, Delete Events
- 🖼️ Upload & Manage Event Images
- 📍 Organized Events by Users
- 🔎 Search & Filter by Keyword, Category, or Date
- 🧾 Secure Checkout with Stripe
- 🛒 Order Management & Tracking
- 🔁 Webhooks for Clerk & Stripe
- 🧠 Related Events Section
- ⚙️ Scalable File Structure & Reusable Components
